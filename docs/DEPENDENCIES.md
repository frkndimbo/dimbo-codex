# Dependencies

## Required

- `git`
- `python3`
- POSIX shell utilities: `cp`, `mkdir`, `find`, `chmod`
- Codex CLI using `~/.codex`

## Recommended

- `rtk` for all interactive shell commands run by agents.
- `uv` or `uvx` for Python tool installs.
- `graphifyy` for the active graph tool.

## Secret Inputs Not Stored Here

- `CONTEXT7_API_KEY`
- `OPENAI_API_KEY` if used by local model provider routing.
- Any API keys, tokens, private keys, auth files, sqlite state, history, or logs.
